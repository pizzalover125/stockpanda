# stockpanda

an open source chess engine built with Python

## why the name?

stockfish is the best chess engine in the world. my last name is panda. i combined the names to come up with "stockpanda"

## features

- minimax
- alpha-beta pruning
- iterative deepening
- transposition tables
- move order
- quiessence search
- opening book
- tablebase
- phase-based evaluation

## planned features

- custom piece / board design
- lichess integration
- cli tool
- custom logo

## technical info

server made with python. client made with Flask.
